# Module 8 Individual Assignment - Interaction Design Guidelines

**Due:** Mon Mar 16, 2026 11:59pm
**Type:** Individual (must be your own work)

## Purpose

Consider new interaction techniques for a common application task and how these might impact the user experience, including mobile and universal design.

## Instructions

1. Select a common application task (e.g., navigating list of items, invoking commands on content, entering formed text)
2. Build a list of alternatives to the standard interaction techniques for this task (e.g., chatbot, AR, VR, brain-computer interfaces, etc.)
3. Describe pros and cons of each alternative
4. Describe how each alternative might be adapted to support mobile and universal design

## Deliverable

Submit as a Word document or PDF on Canvas.
